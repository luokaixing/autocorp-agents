## 二、行业深度分析（10 个行业）

### 行业 1：AI 基础设施与算力（GPU 云、推理服务、模型托管）

- **利润规模**：
  - NVIDIA Q1 FY2026（截至 2026-04-26）季度营收 **816.15 亿美元**，同比 +85%；季度营业利润 **535.36 亿美元**（+147%）；季度净利润 **583.21 亿美元**（+211%）。
  - 全球数据中心以太网交换机市场 Q1 2026 创历史新高 **154 亿美元**，同比 +39.8%，NVIDIA 首次登顶该市场营收榜首。
  - 来源 1：https://m.toutiao.com/group/7656673944660902409/ （WebFetch 已验证）
  - 来源 2：https://www.nvidia.cn/newsroom/ （NVIDIA 官方新闻中心，2026-02-28 发布 FY2026 Q4 及全年财报）
  - 来源 3：http://m.toutiao.com/group/7655949580832260634/ （数据中心交换机市场数据）

- **利润率**：
  - NVIDIA 季度营业利润率 = 535.36 / 816.15 = **65.6%**；净利率 = 583.21 / 816.15 = **71.5%**。
  - 西部数据（WDC）2026 Evercore 会议上披露 AI 关键基础设施增量毛利率（incremental gross margin）高踞 **70%-75%** 区间。
  - 来源：http://m.toutiao.com/group/7651249093821350443/ （WebSearch 已获取）
  - 行业对比：传统制造业毛利率 20-30%，零售业 10-20%，AI 算力毛利率是行业外平均的 2-3 倍。

- **增长率**：
  - NVIDIA 季度营收 YoY +85%，营业利润 YoY +147%，净利润 YoY +211%，仍处高速增长期。
  - NVIDIA FY2027 Q1 展望营收约 780 亿美元（±2%），且不包含中国数据中心收入。
  - 来源：http://m.toutiao.com/group/7654022306565587482/ （WebSearch 已获取）

- **利润池分布**：
  - 上游 GPU 设计与制造（NVIDIA）：吃掉价值链最大份额，毛利率 70%+
  - 中游云算力提供（AWS / Azure / GCP / Oracle Cloud）：转售 GPU 算力，毛利率 30-50%
  - 下游推理服务 / 模型托管（Together / Anyscale / Replicate）：薄利多销，毛利率 20-40%
  - 利润高度集中于"芯片设计 + 算力批发"两环，分销与售后几乎不沾利润。

- **进入壁垒**：
  - **资金壁垒极高**：训练 GPT-5/Claude 4 级别模型需 10-100 亿美元算力投入
  - **技术壁垒极高**：CUDA 生态护城河 + 先进封装产能（台积电 CoWoS）
  - **牌照壁垒中等**：出口管制（美国对华 AI 芯片限制）
  - **网络效应强**：CUDA 开发者生态 + 云厂商锁定

- **AI 公司可切入性**：**低（不推荐）**
  - 已验证能力（英文内容生产 / 浏览器自动化 / 多 agent 编排 / Web 研究）与该行业核心要求（GPU 采购、数据中心建设、CUDA 工程能力）完全不匹配
  - 无资金、无硬件、无工程师团队，结构性不可行

- **3 个机会点**：
  1. 生产 GPU 云厂商对比评测的英文 SEO 长文，通过 affiliate / referral 链接变现（已验证能力：英文内容生产 + Web 研究）
  2. 用 Web 研究能力抓取各大云厂商 GPU 实时价格，输出"AI 算力价格周报"作为内容资产引流
  3. 多 agent 编排能力用于"GPU 现货可用性监控 Agent"，对接开发者社区作为免费工具引流

- **风险**：
  - 监管风险：美国出口管制扩大、中国反制关税（川普 15% 协议已落地）
  - 竞争风险：Broadcom 定制 ASIC（Jalapeño）+ Google TPU + OpenAI 自研芯片正在分流
  - 技术变迁：训练需求见顶后，推理优化（量化、蒸馏）可能压缩算力需求

### 行业 2：AI 应用层（垂直 SaaS、Agent 平台、Coding 助手）

- **利润规模**：
  - Anthropic 年化收入（ARR）2026 年 2 月达 **近 200 亿美元**，较 2025 年底 90 亿美元翻倍，2 周内飙涨 50 亿美元。胡润《2026 全球独角兽榜》Anthropic 估值 6.6 万亿元，成为全球价值最高独角兽。
  - 全球 AI 应用 2026 H1 用户支出预计超 **42 亿美元**，较 2025 H1 的 18.3 亿美元翻倍。
  - 来源 1：http://m.toutiao.com/group/7613574164041040393/ （WebFetch 已验证）
  - 来源 2：http://m.toutiao.com/group/7656964316243116582/ （WebFetch 已验证，Sensor Tower《2026 年 AI 现状报告》）
  - 来源 3：https://cj.sina.cn/article/norm_detail?url=http%3A%2F%2Ffinance.sina.cn%2F2026-06-25%2Fdetail-inierayw5756472.d.html （胡润 2026 独角兽榜）

- **利润率**：
  - Anthropic / OpenAI 具体毛利率未公开披露（**未验证**）
  - 行业基准：AI 应用层 SaaS 毛利率通常 60-75%，但扣除推理成本后净利率可能仅 10-30%
  - Claude 付费转化率约 13%，领跑全行业（Sensor Tower 数据）

- **增长率**：
  - Anthropic 2026 年 2 月美国市场份额飙至 70%（反超 OpenAI），ARR 翻倍
  - ChatGPT 全球市场份额首次跌破 50% 至 46.4%（2026 年 5 月），Gemini 27.7%、Claude 10.3%，三强鼎立
  - AI 应用下载量 2026 H1 预计 23 亿次，用户支出 +130% YoY，但增速开始放缓
  - 来源：http://m.toutiao.com/group/7656964316243116582/ （WebFetch 已验证）

- **利润池分布**：
  - 基础模型层（OpenAI / Anthropic / Google）：吃掉最大份额，但训练成本高昂
  - 应用层（Cursor / GitHub Copilot / Perplexity / Replit）：快速规模化但毛利被推理成本侵蚀
  - 分发渠道（App Store / Google Play / GitHub）：坐收 15-30% 抽成，几乎纯利
  - Coding 助手细分：GitHub Copilot + Cursor + Claude Code 合计覆盖超 1500 万开发者，市场已形成双强格局（Copilot + Cursor）

- **进入壁垒**：
  - **资金壁垒高**：基础模型训练需 10+ 亿美元
  - **技术壁垒高**：需顶级研究团队
  - **应用层壁垒中等**：基于 API 的 wrapper 应用门槛低，但护城河浅
  - **网络效应强**：用户数据飞轮 + 品牌信任

- **AI 公司可切入性**：**中（部分子赛道可行）**
  - 已验证能力（英文内容生产 + Web 研究）可切入"AI 工具评测 / 教程内容"赛道
  - 已验证能力（多 agent 编排）可尝试构建轻量级 Agent 应用，但缺乏工程化能力
  - 浏览器自动化可用于自动化测试 AI 工具并产出对比报告
  - 不可行：基础模型训练、Coding 助手开发（需顶级工程团队）

- **3 个机会点**：
  1. 生产"Claude vs ChatGPT vs Gemini 实测对比"系列英文长文（已验证能力：英文内容生产 + 多 agent 编排可同时调用三家 API 测试）
  2. 用 agent-browser 自动化抓取 ProductHunt / HackerNews 上新发布的 AI 工具，产出"AI 工具周报"内容资产
  3. 多 agent 编排构建"AI Agent 评测平台"内容生产线，每个 agent 专测一类（编程 / 客服 / 数据分析）

- **风险**：
  - 竞争风险极高：OpenAI / Anthropic / Google 三强烧钱抢市场，小玩家易被碾压
  - 技术变迁：模型能力快速提升使 wrapper 应用随时可能被官方功能替代
  - 监管风险：美国政府对 Anthropic 一度"封杀"事件（被神学家联名辩护），政策不确定性高

### 行业 3：加密货币与 Web3（交易所、MEV、流动性质押、RWA）

- **利润规模**：
  - Coinbase Q1 2026 总营收 **14.13 亿美元**（-31% YoY），**净亏损 3.94 亿美元**（由盈转亏），调整后 EBITDA 3.03 亿美元（-67%）。交易业务收入 7.56 亿美元（-40%），订阅与服务收入 5.84 亿美元，稳定币收入 3.05 亿美元（+11%）。
  - 全球 RWA 链上资产总值突破 **250 亿美元**，BlackRock BUIDL 基金规模突破 **17 亿美元**，重点增配亚洲基建与绿色能源资产。
  - 来源 1：http://m.toutiao.com/group/7637544993925743104/ （WebFetch 已验证）
  - 来源 2：http://m.toutiao.com/group/7612633238648914478/ （WebSearch 已获取）
  - 来源 3：https://m.php.cn/faq/2074505.html （Binance 2026 入口信息）

- **利润率**：
  - Coinbase 调整后 EBITDA 利润率 = 303 / 1413 = **21.4%**（Q1 2026，寒冬期）
  - 顶峰期（2024）Coinbase EBITDA 利润率曾达 40-50%
  - Binance 未公开具体利润率，行业估算净利率 30-50%
  - 稳定币发行商（Circle）Q1 2026 财报"喜忧参半"，USDC 流通量与链上交易量增长强劲但战略转型投入大

- **增长率**：
  - Coinbase Q1 2026 营收 -31% YoY，加密寒冬
  - 但 Coinbase 全球现货市场份额升至 **8.6% 创历史新高**，衍生品业务 +169%，稳定币收入 +11%
  - RWA 赛道仍处爆发期，BlackRock BUIDL 从 0 到 17 亿美元仅用 ~1 年
  - 来源：http://m.toutiao.com/group/7637544993925743104/ （WebFetch 已验证）

- **利润池分布**：
  - 交易所手续费（Binance / Coinbase / OKX）：吃掉最大份额，但高度依赖交易量
  - 稳定币发行（Tether / Circle）：几乎纯利的国库券利息收入，Tether 2024 年利润超 130 亿美元
  - MEV / 流动性质押（Lido / Jito / Flashbots）：链上协议层利润，毛利高但规模有限
  - RWA 代币化（BlackRock BUIDL / Franklin Templeton Benji）：起步期，规模小但增长快
  - 链上美债（BUIDL）收益率 4.5-5%，DAI 抵押品配置 RWA 提供 4-6% 年化

- **进入壁垒**：
  - **资金壁垒高**：交易所需储备金 + 合规成本
  - **牌照壁垒极高**：美国 MSB / 纽约 BitLicense / 香港 VASP / 欧盟 MiCA
  - **技术壁垒中等**：链上协议可 fork，但安全审计成本高
  - **网络效应极强**：流动性飞轮（Binance 占据现货 40%+）

- **AI 公司可切入性**：**极低（结构性不可行）**
  - 已验证失败：无法持私钥 → 无法参与链上交易、MEV、流动性挖矿
  - 已验证失败：无法直接收款 → 无法运营交易所或稳定币
  - 已验证失败：记忆过时 + 翻墙依赖 → 无法实时推荐加密机会
  - 已验证失败：3 篇 Paragraph 加密研究文章零打赏 → 内容→打赏闭环已被证伪
  - 参考 `ai-company-capability-boundary-2026-06-30.md` 第六节："彻底暂停加密赛道作为主收入路径"

- **3 个机会点**（仅限内容引流，不直接产生收入）：
  1. 生产"RWA 赛道季度报告"英文长文，对接传统金融读者（已验证能力：英文内容生产 + Web 研究）
  2. 用 Web 抓取能力构建"稳定币收益率实时看板"作为内容引流工具
  3. 多 agent 编排产出"加密交易所透明度月报"（参考 RootData 2026 年 1 月报告框架），纯内容资产

- **风险**：
  - 监管风险极高：美国《清晰法案》推进中、香港稳定币监管、欧盟 MiCA 全面落地
  - 周期性风险：Coinbase Q1 由盈转亏证明加密寒冬随时到来
  - AI 公司结构性冲突：7 项已验证局限中 5 项与加密赛道要求直接冲突（详见 `ai-company-capability-boundary` 第六节）

### 行业 4：医疗健康生物科技（GLP-1、基因治疗、数字疗法）

- **利润规模**：
  - 全球 GLP-1 类药物市场规模 2026 年预计突破 **1000 亿美元**。
  - 诺和诺德 Wegovy 在降低心血管事件风险方面真实世界数据优于礼来 Zepbound（ESC 2025）。
  - 恒瑞医药 2026-06-26 在 Clinicaltrials.gov 注册 HRS9531 片剂（口服 GLP-1/GIP）减重三期临床，计划入组 425 例，预计 2027 年完成。
  - 来源 1：http://m.toutiao.com/group/7657033185527595535/ （WebFetch 已验证）
  - 来源 2：http://m.toutiao.com/group/7655863800511332907/ （恒瑞 GLP-1 三期临床）
  - 来源 3：https://m.familydoctor.cn/news/nuohenuode-wegovy-xinxueguan-fengxian-youyu-zepbo-194268.html （Wegovy vs Zepbound 真实世界数据）

- **利润率**：
  - 诺和诺德净利率长期保持 **30%+**（VestLab 分析：利润率高得吓人）
  - 礼来净利率同样 25-30%
  - GLP-1 制剂毛利率通常 80%+（生物药制剂特性）
  - 来源：https://xueqiu.com/8137218214/376617261 （VestLab 财报分析）

- **增长率**：
  - GLP-1 市场年增长率 25-35%（2024-2027）
  - 偏向型 GLP-1（先维盈® 埃诺格鲁肽，辉瑞中国 + 先为达生物）SLIMMER Ⅲ期数据：48 周最高剂量组体重下降 15.4%，92.8% 受试者实现 5%+ 体重减轻
  - 中国 GLP-1 减重注射液市场多家厂商布局：诺和诺德、礼来、华东医药、博瑞医药、恒瑞、信达生物、仁会生物

- **利润池分布**：
  - 专利药企（诺和诺德 / 礼来）：吃掉最大份额，毛利率 80%+
  - 分销渠道（药房 / 医院）：毛利率 10-20%
  - 仿制药企（华东医药 / 博瑞 / 恒瑞）：利润薄但规模大
  - 数字疗法 / 生活方式干预配套：利润池正在形成，但规模小
  - 医美 / 减重服务（诊所 / 在线平台）：服务毛利 30-50%

- **进入壁垒**：
  - **资金壁垒极高**：新药研发 10-30 亿美元
  - **技术壁垒极高**：分子设计 + 三期临床 + FDA / NMPA 审批
  - **牌照壁垒极高**：药品上市许可、GMP 生产许可
  - **专利壁垒极强**：诺和诺德 / 礼来专利保护至 2030+

- **AI 公司可切入性**：**极低（核心赛道不可行）**

- **利润池分布**：
  - 平台（亚马逊 / Shopee / Temu）：抽佣 8-15% + 广告 + 物流，利润最丰厚

  - 卖家（品牌 / 白牌）：毛利率 15-50%，差异大
  - 物流（递四方 / 燕文 / 极兔）：毛利率 10-20%，规模驱动
  - 退货 / RMA 外包（如珠海楚海客 7200 人团队）：毛利率 15-25%

  - **资金壁垒中等**：独立站启动成本低（Shopify 月费 $29 起），但流量获取成本高

  - **牌照壁垒低**：海外公司注册 + 税号即可

### 3.2 关键结论

**Top 5 全部为"内容 + 工具"切入模式**，无一是"产品 / 平台"模式。这反映了 AI 公司已验证能力（英文内容生产 + 浏览器自动化 + 多 agent 编排 + Web 研究）的本质：**内容生产能力是当前唯一已验证可直接变现的能力**，工具侧（多 agent 编排）可作为内容资产附属引流，但独立变现路径未验证。

**Top 1（企业服务 SaaS 内容）与 Top 2（电商内容）** 的共同特征：
- 与已验证能力（英文长文 + Web 研究）直接匹配
- 目标读者（企业 IT 决策者 / 独立站卖家）付费意愿高
- 高 CPM 关键词（SaaS / Shopify SEO）适合 Google Ads 变现
- 与已发布 Paragraph 文章能力同源，可立即启动
