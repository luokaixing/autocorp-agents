## 发布操作步骤（每篇）

### 1. 准备内容
1. 打开文件：`d:\autoCompany\company\projects\seo-content-generator\articles\XX-文件名.md`
2. 用记事本打开
3. 全选 + 复制（Ctrl+A → Ctrl+C）

### 2. 在 Publish0x 创建文章
1. 登录 https://www.publish0x.com
2. 点击「Write a story」或「New Story」
3. 在编辑器中粘贴内容（Ctrl+V）
4. 检查格式是否正确（标题、段落、列表）

### 3. 设置文章信息
- **标题**：从上方表格复制
- **标签**：从上方表格复制（最多 5 个，用逗号分隔）
- **封面图**：从 Unsplash 找一张相关图片（免费）
  - 网址：https://unsplash.com
  - 搜索关键词：crypto, blockchain, bitcoin, ethereum
- **摘要**：取文章第一段前 150 字符

### 4. 发布
1. 点击「Preview」预览
2. 检查格式无误后点击「Publish」
3. 等待审核（通常 24 小时内通过）

## 优化技巧

### 提高打赏的技巧
1. **文章开头**添加一段个人介绍：
   ```
   *By AutoCorp Insights - AI-powered crypto tutorials. 
   If this helped you, a tip would be appreciated! 🙏*
   ```
2. **文章末尾**添加打赏说明：
   ```
   ---
   ## Enjoyed this guide?
   If you found this helpful, please use the "Tip" button below to support the author.
   You can also tip directly to: `${ETH_TIPPING_ADDRESS}`
   ```
3. **互动**：回复每一条评论
4. **关注**：关注其他 crypto 作者，他们可能回关你

### Publish0x 打赏机制
- 平台每天发放打赏池（约 $50-100）
- 读者打赏 = 80% 给作者 + 20% 给读者
- 读者不花自己的钱，所以打赏意愿高
- 你也可以读别人的文章赚打赏（双赢）

## 注意事项

1. **不要一天发多篇**：会被判定 spam，账号可能被封
2. **内容必须原创**：我们的文章是 AI 生成，属于原创
3. **不要在文章中放外部链接**：除非是引用来源
4. **标签要准确**：错误的标签会影响曝光
5. **封面图必须免费授权**：用 Unsplash 或 Pexels
