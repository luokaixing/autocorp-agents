## 赛道 1：AI 内容再利用工具

长内容(播客/YouTube 长视频/博客)→ 短视频脚本、TikTok/Reels/LinkedIn 帖子。

### 痛点
- 内容创作者每周需花费 5-10 小时手工把长内容拆解、改写为多平台短内容,产出效率极低。
- 跨平台格式差异大(TikTok 钩子、LinkedIn 专业口吻、Twitter 短链),人工适配易疲劳出错。
- 现有工具多停留在「转录切片」,缺乏「理解语义后重新生成」的能力,内容质量参差。

### 目标用户
- YouTube 博主(1 万-50 万订阅,有变现但缺团队)
- 自媒体运营者、Newsletter 作者
- 中小企业营销团队(2-10 人,需多平台分发但人手不足)
- Podcast 主理人

### 竞品分析
- **Repurpose.io**:主打自动化分发,但内容改写能力弱,定位偏工具层。
- **Opus Clip**:短视频切片赛道头部,融资充足,但价格偏高且对中文/小语种支持一般。
- **Castmagic / Deciphr**:播客方向,聚焦转录+素材,对「再创作」支持有限。
- 差异化机会:多模态理解(音视频+字幕)+ 风格化改写(模仿创作者个人语调)。

### 变现模型
- SaaS 订阅制:$19/月(Starter,单平台)/ $39/月(Pro,多平台)/ $49/月(Business,团队席位)
- 年付优惠 2 个月,提升 LTV。
- 可叠加:按分钟计费的额外渲染额度包。

### 预期 MRR
- $5,000-15,000(冷启动 6 个月内可达下沿,12 个月冲击上沿)
- 假设:平均客单价 $35/月,150-430 个付费用户。

### 技术可行性（AI 自主开发难度）
- **中等**。核心依赖:ASR(可直接调 Whisper API / Deepgram)、LLM 改写(Claude/GPT)、可选 TTS+口播数字人。
- AutoCorp 现有 Agent 编排能力可直接复用,前端为标准 Next.js + 支付(Stripe)。
- 风险点:视频渲染管线(ffmpeg + 云函数)工程量较大,可先用纯文本脚本输出 MVP 规避。

### 推荐指数
★★★★☆

## 赛道 2：SEO 内容自动化生成

### 痛点
- SEO 内容生产慢:一篇高质量长文从关键词研究到成稿通常 4-8 小时。
- 关键词研究门槛高:小企业主不懂工具(Ahrefs/SEMrush 月费 $100+),难以发现低竞争长尾词。
- 通用 AI 生成的 SEO 文章同质化严重,Google 2024-2025 算法更新后对「低质 AI 内容」打击明显,需要 EEOP(经验/专业性/权威性/可信度)信号。

### 目标用户
- 独立站长、Affiliate 从业者
- 小企业主(本地服务、电商独立站)
- SEO 机构、内容工作室(需规模化产出)
- SaaS 公司内容营销团队

### 竞品分析
- **Surfer SEO**:客单价高($69-249/月),偏 NLP 优化,需人工写作配合。
- **Frase**:关键词+大纲+生成一体,但 2025 年被收购后路线图不稳。
- **Byword.ai / Koala.sh**:批量生成方向,价格战激烈,质量波动大。
- 差异化机会: SERP 实时分析 + 实体图谱 + 程序化 SEO(PSEO)批量站点生成。

### 变现模型
- 按篇计费:$1-5/篇(批量,吸引试水用户)
- 订阅制:$29/月(50 篇)/ $79/月(200 篇)/ $99/月(无限制 + PSEO)
- 附加:关键词研究额度包、内部链接自动化插件。

### 预期 MRR
- $8,000-20,000
- 假设:平均客单价 $49/月,160-400 个付费用户;按篇付费用户转化漏斗顶部更大。

### 技术可行性（AI 自主开发难度）
- **中高**。核心依赖:SERP 抓取(需代理池/反爬)、关键词难度评估模型、LLM 结构化生成、CMS 发布集成(WordPress/Webflow)。
- 程序化 SEO 部分需要模板引擎 + 大规模静态站点生成,工程量中等偏上。
- AutoCorp 优势:Agent 可自主完成关键词聚类、内容大纲、内链策略全流程。

### 推荐指数
★★★★★

## 赛道 3：AI Chrome 扩展（生产力类）

### 痛点
- 知识工作者每天在浏览器内重复操作:跨标签页整理信息、摘要长文、邮件回复、网页翻译与提炼。
- 通用 ChatGPT 网页版需切换上下文,无法直接「读取当前页面」。
- 现有扩展多为单点功能(只摘要/只翻译),缺乏统一工作流。

### 目标用户
- 知识工作者、咨询顾问、分析师
- 研究人员、研究生
- 销售/BD(需快速理解客户网站)
- 内容创作者(素材收集)

### 竞品分析
- **Monica**:全功能型,用户基数大,定价 $9.9-39/月。
- **Sider**:侧边栏交互,多模型切换,融资充足。
- **Merlin**:类似定位,运营激进出海。
- 差异化机会:垂直工作流(如「研究 → 笔记 → Notion 同步」一键链路)、隐私优先(本地小模型)。

### 变现模型
- Freemium:免费每日 20 次,Pro $9.9/月(无限 + 高级模型)
- 团队版:$19/月/席位(共享 prompt 库 + 管理后台)
- 按量:超出额度后按 token 计费。

### 预期 MRR
- $3,000-10,000
- 假设:平均客单价 $9.9/月,300-1000 个付费用户(免费用户基数可达数万)。

### 技术可行性（AI 自主开发难度）
- **中等**。Chrome 扩展 Manifest V3 开发成熟,难点在:多模型 API 路由、上下文长度管理、跨标签页 DOM 提取的稳定性。
- 商业化难点:Chrome Web Store 审核周期长、获客成本高(需 SEO + ProductHunt + 网红)。
- AutoCorp 自主开发 MVP 可 2 周内完成,但增长是更大瓶颈。

### 推荐指数
★★★☆☆

## 赛道 4：垂直行业 AI Agent（客服 / 数据分析 / 法律文书）

### 痛点
- 垂直场景下通用 AI(ChatGPT/Claude 网页版)不够专业:不懂行业术语、无法接入业务系统、缺乏合规校验。
- 中小企业自建 Agent 缺技术能力,外包定制动辄 $20k+ 起步。
- 现有垂直 SaaS(如传统客服系统)智能化程度低,响应慢、转人工率高。

### 目标用户
- 中小企业(电商、SaaS、本地服务)客服团队
- 律所、合规部门(合同审查、文书起草)
- 电商运营(数据分析、报表自动化)
- 医疗诊所前台(预约、问答)

### 竞品分析
- **Intercom Fin**:客服 Agent 头部,客单价高($0.99/次解决),适合中大型。
- **Harvey**:法律 AI 独角兽,估值高,只服务大所。
- **Sierra**(Bret Taylor 创办):企业级客服 Agent,融资巨量。
- 差异化机会:面向 SMB 的「开箱即用 + 可定制」Agent,价格仅为头部 1/5,聚焦单一垂直(如 Shopify 电商客服)。

### 变现模型
- 按席位:$49/月(标准)/ $199/月(高级,含自定义知识库 + API)
- 按对话量:前 1000 次/月免费,超出 $0.1/次
- 实施费:一次性 $500-2000(知识库搭建、系统集成)

### 预期 MRR
- $10,000-30,000
- 假设:平均客单价 $99/月,100-300 个付费席位;客单价高、流失率低,LTV 极佳。

### 技术可行性（AI 自主开发难度）
- **高**。需 RAG 知识库、工具调用(对接 Shopify/Zendesk/Slack)、多轮对话状态管理、人工接管、合规审计日志。
- 单一垂直(如电商客服)MVP 可 4-6 周完成,但护城河来自行业 know-how 沉淀。
- AutoCorp 优势:多 Agent 协作能力天然适配复杂工作流。
- 风险:客单价高意味着销售周期长(2-8 周),初期现金流压力大。

### 推荐指数
★★★★★

## 赛道 5：Landing Page / 落地页生成器

### 痛点
- 非技术人员做落地页难:现成模板千篇一律,自定义又需写代码。
- 做落地页慢:从设计到上线常需 1-3 天,营销活动节奏跟不上。
- A/B 测试与转化优化门槛高,小团队无力持续迭代。

### 目标用户
- 独立开发者、Indie Hacker
- 小创业团队(2-5 人)
- 营销人员、增长黑客
- 自由职业设计师(为客戶快速交付)

### 竞品分析
- **Carrd**:极简单页,$19/年,用户基数巨大但功能有限。
- **Framer**:设计感强,但学习曲线陡,价格 $5-30/月。
- **Lovable / v0.dev**:AI 生成代码方向,资本关注度高,但产出仍需调试。
- 差异化机会:聚焦「营销转化」而非「自由设计」——内置文案生成、转化文案模板、自动 A/B 测试、CRM 集成。

### 变现模型
- 订阅制:$15/月(单站点)/ $39/月(多站点 + 自定义域名 + A/B 测试)
- 一次性:模板市场分润
- 附加:域名注册、托管(年付)

### 预期 MRR
- $5,000-12,000
- 假设:平均客单价 $25/月,200-480 个付费用户。

### 技术可行性（AI 自主开发难度）
- **中低**。核心:LLM 生成文案 + 模块化组件库 + 拖拽编辑器 + 静态托管(Cloudflare Pages/Vercel)。
- 拖拽编辑器是工程量大头,可用开源(Grapesjs/Plasmic)二次开发降低成本。
- AutoCorp 优势:Agent 可根据用户一句话需求自主生成完整页面 + 文案 + SEO 元信息。

### 推荐指数
★★★★☆

## 赛道 6：AI 简历 / 求职工具

### 痛点
- 求职者简历优化难:不懂 ATS( applicant tracking system )关键词匹配,投递石沉大海。
- 面试准备无方向:缺少针对 JD 的模拟面试与反馈。
- 美国市场求职竞争激烈,尤其 2026 年科技行业回暖后求职者付费意愿强。

### 目标用户
- 求职者(应届生、转行者、被裁科技从业者)
- 美国市场尤其活跃(H1B、绿卡申请者)
- 职业教练、大学就业指导中心(B2B)

### 竞品分析
- **Teal**:简历构建 + JD 匹配,$9-29/月,用户基数大。
- **Rezi**:ATS 优化方向,$3-29/月,韩国团队出海美国。
- **Jobscan**:关键词匹配老牌,$49.95/月。
- 差异化机会:AI 模拟面试(语音)+ 简历动态优化 + Offer 谈判辅导的全链路。

### 变现模型
- Freemium:免费简历分析,Pro $9/月(简历无限 + JD 匹配)/ $29/月(含模拟面试)
- 一次性:简历精修服务 $49-99(人工 + AI 协作)
- B2B:大学/训练营批量授权

### 预期 MRR
- $4,000-10,000
- 假设:平均客单价 $15/月,270-670 个付费用户;季节性强(校招季峰值)。

### 技术可行性（AI 自主开发难度）
- **中低**。核心:简历解析(PDF/Word)、JD 关键词提取与匹配、LLM 改写、可选语音模拟面试(Whisper + TTS)。
- ATS 兼容性渲染(PDF 格式把控)是细节难点。
- AutoCorp 优势:Agent 可全流程自主完成「分析 → 优化 → 模拟面试 → 反馈」。

### 推荐指数
★★★★☆

## 决策建议（供 CEO 参考）

### 第一阶段(0-3 个月):现金流优先
启动 **赛道 2(SEO 内容自动化)** + **赛道 1(AI 内容再利用)**:
- 技术栈高度复用(LLM 调用 + Web 前端 + Stripe)
- 目标:3 个月内合计 MRR 达 $8,000

### 第二阶段(3-6 个月):客单价与护城河
切入 **赛道 4(垂直 AI Agent,电商客服方向)**:
- 复用第一阶段的 RAG 与 Agent 编排能力
- 目标:6 个月内新增 MRR $10,000

### 第三阶段(6-12 个月):产品矩阵
按需扩展 **赛道 5(落地页)** 或 **赛道 6(求职工具)** 作为流量入口或季节性现金流补充。

### 风险提示
- 通用 LLM 能力下沉可能导致工具型赛道(3、6)被免费替代。
- 高客单价赛道(4)销售周期长,需预留 6 个月 runway。
- 所有赛道均需关注数据合规(GDPR/CCPA),尤其用户内容类(1、2、6)。
