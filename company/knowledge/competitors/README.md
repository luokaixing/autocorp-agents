# 竞品分析（Competitors）

> 用途：记录 AutoCorp 关注的竞争对手动态

本目录用于持续跟踪各赛道竞争对手的产品、定价、融资、功能变化,为决策提供情报支撑。每个竞品建议记录:

- **公司/产品基本信息**:名称、官网、团队规模、融资阶段
- **核心产品与定价**:主要功能矩阵与价格档位
- **优势与劣势**:客观评价
- **最近动态**:功能更新、融资、人事变动(附日期)
- **对 AutoCorp 的启示**:可借鉴 / 需规避之处

文件组织建议:按赛道分子目录,如 `seo-content/surfer-seo.md`、`ai-agent/intercom-fin.md`。

> 竞品情报每季度至少更新一次,重大动态(融资、收购、重大功能发布)即时归档。
